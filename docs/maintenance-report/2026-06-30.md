## Code Quality

- **Bug fixed**: `index.html`'s Projects section had an unclosed `<div data-aos="zoom-in">` (opened line 397, originally never closed before `</section>`). Browsers silently auto-close this on `</section>`, so it caused no visible rendering issue, but it was invalid markup that would fail any stricter parser/validator. Added the missing `</div>`.
- **Lint fixes**: three unquoted attribute values (`data-natural-width`, `data-natural-height`, `data-position-y` in `index.html`) and one single-quoted attribute pair (`rel`/`href` on the Google Fonts link in `404.html`) were corrected to double-quoted, per the new HTMLHint check.
- **Dead code removed**: 12 unused, fully-vendored Bootstrap 4.6.0 files (~2.0 MB) — see Dependency Audit.
- No duplicate logic, unused variables, or significant technical debt identified beyond the above. The codebase is a small, hand-maintained static site; no further refactoring was performed (per "avoid unnecessary refactoring" policy).
